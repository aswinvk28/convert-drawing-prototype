# Convert Drawing App

Convert Drawing is a test app that runs on native Javascript. The dev packages currently used are:

- "body-parser": "~1.18.2",
- "cookie-parser": "~1.4.3",
- "debug": "~2.6.9",
- "express": "~4.15.5",
- "jade": "~1.11.0",
- "morgan": "~1.9.0",
- "serve-favicon": "~2.4.5"
